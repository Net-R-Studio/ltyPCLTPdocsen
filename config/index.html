# Docs

Language differences

Since Plain Craft Launcher 2 only supports Chinese, it needs to be used with the help of translation.

First, you need to sponsor the PCL2 author LTCat to download the sponsored version of PCL2. LTCats afdian (opens new window)
Run PCL2 to unlock the custom theme.
Exit PCL2 after unlocking.
Delete the files in PCL and unzip them to this folder.
Start PCL2 again and the installation is complete.
